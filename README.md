# 🎧 Ultimate Ears Website Clone

This is a **frontend clone of the Ultimate Ears website** built using **HTML and CSS**.  
The project focuses on recreating the layout and design of the original website to practice frontend development.

## 🌐 Live Demo
https://ultimate-ears-in.netlify.app/

## 🚀 Features
- Responsive website layout
- Modern UI design
- Navigation bar
- Hero section
- Product showcase section
- Footer

## 🛠️ Tech Stack
- HTML5
- CSS3

## 📂 Project Purpose
This project was created for **learning and practice purposes** to improve HTML and CSS skills by cloning a real-world website.
